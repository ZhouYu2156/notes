

# Snake Game

## Introduction

> This is a Snake Eating Game.


## Usage
- First: 

> Download project to local location

- Second:

> Execute command to install project dependencies
>
> ```bash
> $ npm i     # It is equivalent to `npm install`
> ```



## Project `StartUp`

```bash
$ npm run dev
```

## Project `Build`
```bash
$ npm run build
```
